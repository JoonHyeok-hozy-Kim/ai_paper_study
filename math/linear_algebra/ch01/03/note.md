# 1.3 SYSTEMS OF LINEAR EQUATIONS

#### Def) Linear Equations
* $a_1x_1+a_2x_2+...+a_nx_n = b$
  * where $x_1, x_2, ..., x_n$ are variables
  * and $a_1, a_2, ..., a_n$, and $b$ are real numbers.
    * $a_1, a_2, ..., a_n$ : coefficients
    * $b$ : constant term
