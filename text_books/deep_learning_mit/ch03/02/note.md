# 3.2 Random Variables

### Concept) Random Variable
- Def.)
  - A variable that can take on different values randomly
- Notation)
  - $x$ : a random variable
    - $x_1, x_2, \cdots$ : possible values that $x$ can take on
  - $\mathbf{x}$ : a vector valued random variable
