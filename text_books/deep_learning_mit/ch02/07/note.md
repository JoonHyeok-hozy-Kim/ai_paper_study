# 2.7 Eigendecomposition

Why needed?)
- By decomposing integers into prime factors, we can conclude useful properties.
  - e.g.) $12 = 2^2\times 3$
    - Then
      - $12$ is not divisible by $5$
      - Any integer multiple of $12$ will be divisible by $3$.
